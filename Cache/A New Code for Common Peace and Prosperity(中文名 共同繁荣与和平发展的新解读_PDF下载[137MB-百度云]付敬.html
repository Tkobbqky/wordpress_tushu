作者简介

[

付敬，资深记者，《中国日报》欧洲分社副社长、中欧关系专栏作家，中国观察智库执行主任。曾获中国新闻奖一等奖、中直机关五一劳动奖章称号、长江韬奋奖提名。2010年7月开始派驻比利时布鲁塞尔，对气候变化、环境问题和金融危机等话题进行跟踪报道，参与了2009年伦敦二十国集团峰会、纽约联合国气候变化峰会、匹兹堡二十国集团金融峰会以及哥本哈根气候峰会的系列报道。著有《Testing Time (中欧关系见证录)》，编辑、合著了《绿动中国》(中英文版)、《踢除旧习——联合国气候变化导则》(中文版)、《无法撼动的四川：汶川大地震年鉴编年史》(英文版)、中国政府2009年对外经济宣传册《中国经济绿色恢复》(中英文版)等作品。还分别于2005年、2008年以及2010年在联合国开发计划署《中国人类发展报告》的出版工作中担任顾问，负责该报告的推广。于1998年和2000年分别获得了首都师范大学和中国新闻学院的学士学位，而后又获得了英国伦敦城市大学国际新闻学硕士学位。
