作者简介

[

王汝南，职业八段棋士，中国围棋协会主席。1946年9月出生于安徽合肥。著名的围棋运动员，1966年曾战胜日本九段，1982年被定为八段棋士。20世纪80年代后逐渐转为教练员和管理者，享受国务院津贴，曾任中国棋院第二任院长等，现任中国围棋协会主席。著有《弈坛争霸三十年》《玄玄棋经新解》《中盘攻防指南》等。
---------
尹小林，首都师范大学电子文献研究所所长，首都师范大学国学传播研究院执行院长，长期致力于古籍数字化，古代文献学、辞书学研究。1993年独立编撰《汉语数目词辞典》（中华书局出版），1998年独立开发《全唐诗》检索系统（商务印书馆出版）。大型中华古籍全文检索数据库《国学宝典》的创始人，国学网的创始人。2000年创办的国学网，是以弘扬中国传统文化为根本宗旨的大型学术性公益网站。

]

胜固欣然，败亦可喜。——棋经箴言总论 / 008围棋初非人间之事：始出于巴邛之橘，周穆王之墓；继出于石室，又见于商山，乃仙家养性乐道之具。 / 016习射、习御、习琴、习弈，终无一事可以息得者。 / 024举棋不定，不胜其耦。 / 032博弈好饮酒，不顾父母之养，二不孝也。 / 040以智力求者，譬如弈棋，进退取与，攻劫放舍，在我者也。 / 048行一棋不足以见智，弹一弦不足以见悲。 / 056围棋击剑，反目眩形，亦皆自然也。 / 064世有围棋之戏，或言是兵法之类也。 / 072局必方正，象地则也；道必正直，神明德也；棋有白黑，阴阳分也；骈罗列布，效天文也。 / 080弈之机，虚实是也。 / 088守规不固兮，为所唐突。深入贪地兮，杀亡士卒。狂攘相救兮，先后并没。上下杂沓兮，四面隔闭。 / 096苍天如圆盖，陆地似棋局，世人黑白分，往来争荣辱。 / 104二斗共生，皆目并也；持棋合围，连理形也。 / 112局不弘席，子不盈卷，秉二仪之极要，握众巧之至权。 / 120夫弈者之思，尽于一局者也；圣人之明，周于天下者也。苟一局之势未尝尽同，则天下之事岂必相袭哉？ / 128棋于廉勇之际，得之深矣。且吾未为时用，博弈可也。 / 136虽复理生于数，研求之所不能涉，义出乎几，爻象未之或尽。 / 144围棋有手谈、坐隐之目，颇为雅戏，但令人耽愦，废丧实多，不可常也。 / 152是以须先据四道，守角依傍。 / 160方若棋盘，圆若棋子，动若棋生，静若棋死。方若行义，圆若用智，动若聘材，静若得意。 / 168人心无算处，国手有输时。 / 176不得贪胜、入界宜缓、攻彼顾我、弃子争先、舍小就大、逢危须弃、慎勿轻速、动须相应、彼强自保、势孤取和。 / 184人事三杯酒，流年一局棋。青山不厌千杯酒，白日惟消一局棋。 / 192不害则败，不诈则亡，不争则失，不伪则乱。 / 200仁则能全，义则能守，礼则能变，智则能兼，信则能克。 / 208肥边易得，瘦肚难求。思行则往往失粘，心粗则时时头撞。休夸国手，谩说神仙，赢局输筹即不问，且道黑白未分时，一着落在什么处？ / 216莫将戏事扰真情，且可随缘道我赢。战罢两奁分白黑，一枰何处有亏成？ / 224心似蛛丝游碧落，身如蜩甲化枯枝。 / 232宁输数子，勿失一先。 / 240胜不言，败不语。据廉让之风者君子也，起忿怒之色者小人也。高者无亢，卑者无怯。 / 248一子贵千金，一路重千里。精思入于神，变化胡能拟。 / 256但当偿所负，然诺重嵩华。 / 264懒偿诗债缘何事，熟读棋经要异闻。 / 272君莫笑，闲忙得棋势。 / 280观棋不语真君子，落子无悔大丈夫。 / 288缓着应知心路远，急围不放耳根闲。 / 296十年学弈天机浅，技不能高谩自娱。 / 304有星辰分布之序，有风云变化之机，有春生秋杀之权，有山河表里之势。世事之升降，人事之盛衰，莫不寓是。 / 312看来总是争闲气，笑杀傍观袖手人。 / 320眼前富贵一枰棋，身后功名半张纸。 / 328儒家谓之权，兵家谓之奇，棋家谓之行。 / 336愿尔逢人权放着，世间万事忌孤高。 / 344围棋有五得：得好友，得人和，得教训，得心悟，得天寿。 / 352实实虚虚之同，正正奇奇之妙，此唯审于弃取之宜，明于彼此缓急之情。 / 360若非劲敌，虽胜亦乏精采。求其两相对垒，年力相当，各极所长。 / 368行乎当行，止乎当止，任其自然而与物无竞。 / 376化机流行，无所迹向，百工造极，咸出自然。 / 384以心制数数无穷，以数写心心无尽，生今之时，为今之弈，后此者又安知其不愈出愈奇。
